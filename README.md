## Get Started

```bash
git clone https://github.com/fakiolinho/phonebook.git
cd phonebook
npm i && npm start
```

You also need a `.env` file in the following format (it is ignored by git):

```bash
MONGO_URI=mongodb://localhost/phonebook

HOST=localhost
PORT=3000
```
